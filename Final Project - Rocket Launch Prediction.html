Introduction¶

Objective. The purpose of this final project is to test our cumulative knowledge and learning from the entire semester. We have gained some basic knowledge in physics and programming in Python that we will apply to a rocket launching scenario to facilitate estimations of both max flight height and landing distance. Given a small rocket, our goal is to predict the max height that the rocket reaches and how far from its launch point it will land on the ground.

Methods¶

Experimental Design. Utilizing our experience both in solving physics problems and coding which we have developed this semester, our experiment centered on Python classes, uncertainty values, and Eulers method to produce a computer estimation of our rockets launch path and thus maximum height and distance.

Assumptions. We were assuming the average wind speed in Rexburg to be $3.1 \frac{m}{s}$ and that it will blow at a consistent velocity, however we decided to use the days forecast of a wind speed of $3.6 \frac{m}{s}$. We also assumed the value of acceleration due to gravity $g$ to be $9.81853 \frac{m}{s^2}$. Another assumption we made is that the class A rocket engines are the weakest and will propel the rocket to a height of 50-70 meters. We assumed that the launch point and the landing point would both be at the same elevation.

Algebraic modeling. The variables above were found by taking measurements of the equipment provided or by using online sources and information where applicable. Since the effect due to wind is an element of drag, we included the drag equation in the x direction. The influences included in the vertical distance are thrust, air drag, and the weight of the rocket (which is its mass times acceleration due to gravity).

Before parachute deployment:

$$ F_{net_y} = engineThrust - airDrag - rocketWeight $$

$$ F_{net_x} = windSpeed $$

After parachute deployment:

$$ F_{net_y} = parachuteDrag - rocketWeight $$

$$ F_{net_x} = windSpeed $$

Computational modeling. Using classes in Python, we modeled the rocket as individual components of the rocket engine and rocket body. This allows us to update the mass of the engine and rocket system as time passes and the engine burns. In addition, we can increment time and output chosen variables at any given time. In addition, we included classes that represent drag and uncertainty to facilitate calculations. In short, we used a modular approach to developing our simulation. The simulator is a composite function of the discrete components that make up a launch. (See Code Section for details)

Random Value Sampling for Uncertainties¶

The following Python class represents a random sampling from an uncertain value space, and is designed to be consumed in the code like a float would be (by use of the cannoncial mathematical operators). Though the class is rather verbose, a quick observation will conclude that this class simply takes a random sample given a value and its uncertainty, and stores it in an operator-compatible interface.

from random import uniform

class UncertainValue:
    def __init__ (self, value = 0 , uncertainty = 0):
        self.v = value
        self.u = uncertainty
        self.resample()
        
    def resample(self):
        self.value = uniform(self.v - self.u, self.v + self.u)
        return self.value
        
    def get(self):
        return self.value

    def __float__(self):
        return self.get()
    
    def __radd__(self, other):     return other + self.get()
    def __rmul__(self, other):     return other * self.get()
    def __rlt__(self, other) :     return other < self.get()
    def __rge__(self, other) :     return other >= self.get()
    def __rsub__(self, other):     return other - self.get()
    def __rmod__(self, other):     return other % self.get()
    def __rtruediv__(self, other): return other / self.get()
    def __rle__(self, other):      return other <= self.get()
    def __req__(self, other):      return other == self.get()
    def __rne__(self, other):      return other != self.get()
    def __rgt__(self, other):      return other > self.get()
    
    def __add__(self, other):
        if isinstance(other, self.__class__):

Representing the Engine¶

In order to get a self-contained representation of the class A8-3 engine, we will use a Python class to keep track of the engine state. All values come from this specification.

class Engine:
    def __init__(self):
        self.time             = 0.0
        self.running          = True
        self.BURN_TIME        = UncertainValue(0.7, 0.05)       # s
        self.TOTAL_MASS       = UncertainValue(0.0167, 0.0002)  # kg
        self.FUEL_MASS        = UncertainValue(0.0033, 0.0002)  # kg
        self.IMPULSE          = UncertainValue(9.7, 0.0001)     # N
        self.BURN_RATE        = self.FUEL_MASS / self.BURN_TIME
        self.OUT_OF_FUEL_MASS = self.TOTAL_MASS - self.FUEL_MASS
    
    def getMass(self):
        if not self.running:
            return self.OUT_OF_FUEL_MASS
        else:
            return self.TOTAL_MASS - self.BURN_RATE * self.time
        
    def getThrust(self):
        if not self.running:
            return 0
        else:
            return self.IMPULSE
    
    def incTime(self, value):
        self.time += value
        self.__updateDependants(value)
        
    def __updateDependants(self, dt):
        if (self.time >= self.BURN_TIME):
            self.running = False

        self.value = C * A * .5 * p

Representing the Rocket¶

The rokect has an engine, a parachute, and its own weight. We will keep track of this using another Python class.

class Rocket:
    def __init__(self, windSpeed = 0.0):
        self.engine                         = Engine()
        self.time                           = 0.0
        self.windSpeed                      = windSpeed # m/s 
        self.PARACHUTE_CROSS_SECTIONAL_AREA = UncertainValue(0.0415,  0.0002)      # m^2
        self.TOP_CROSS_SECTIONAL_AREA       = UncertainValue(0.00048, 0.0001)      # m^2
        self.SIDE_CROSS_SECTIONAL_AREA      = UncertainValue(0.018,   0.0001)      # m^2
        self.mass                           = UncertainValue(0.0509,  0.0002)      # kg
        self.position                       = XY()
        self.velocity                       = XY()
        self.acceleration                   = XY(
            0, self.engine.IMPULSE/self.getMass() - CONST.g
        )
        
    def parachuteHasDeployed(self):
        return not self.engine.running
    
    def getXDrag(self):
        if self.parachuteHasDeployed():
            drag = Drag(A = self.SIDE_CROSS_SECTIONAL_AREA)
            return drag.value / self.getMass()
        else:
            return 0.0
    
    def getYDrag(self):
        if self.parachuteHasDeployed():
            drag = Drag(A = self.PARACHUTE_CROSS_SECTIONAL_AREA)
        else:
            drag = Drag(A = self.TOP_CROSS_SECTIONAL_AREA)
        return drag.value / self.getMass()
    
    def getMass(self):
        return self.mass + self.engine.getMass()
    
    def incTime(self, value):
        self.time += value
        self.__updateDependants(value)
        
    def __updateDependants(self, dt):
        self.engine.incTime(dt)
        
        magnitude = np.sqrt(self.velocity.x**2 + self.velocity.y**2) * dt
        drag_part_y = - self.getYDrag() * magnitude
        self.acceleration.y = self.engine.getThrust() / self.getMass() - CONST.g
        
        if (self.velocity.x < self.windSpeed):
            drag_part_x =   self.getXDrag() * magnitude
            self.velocity.x += dt * self.acceleration.x + drag_part_x * self.windSpeed
        else:
            self.velocity.x = self.windSpeed
        self.velocity.y += dt * self.acceleration.y + drag_part_y * self.velocity.y
        self.position.x += dt * self.velocity.x
        self.position.y += dt * self.velocity.y

Simulator¶

This class runs a series of trials on our rocket launch and prints out the results.

class RocketSimulator:
    def __init__(self, numTrials = 0, dt = .01, windSpeed = UncertainValue(3.6, .5)):
        self.dt        = dt
        self.numTrials = numTrials
        self.windSpeed = windSpeed
        
    def resampleValues(self):
        self.windSpeed.resample()
        
    def printTrialReport(self, trialNumber, x, y):
        print ('TRIAL {} of {} | windspeed {:.2f}m/s'.format(trialNumber, self.numTrials, self.windSpeed.get()))
        print ('Max X: {:.2f}m'.format(max(x)))
        print ('Max Y: {:.2f}m'.format(max(y)))
        pyplot.plot(x,y)
        pyplot.show()
        
    def runTrial(self, number, showReport = False):
        self.resampleValues()
        rocket = Rocket(windSpeed = self.windSpeed)
        x = [0]
        y = [0]

        fail_safe = 0
        while (fail_safe < 10000 and rocket.position.y >= 0):
            rocket.incTime(self.dt)
            x.append(rocket.position.x)
            y.append(rocket.position.y)
            fail_safe += 1
        
        if showReport: self.printTrialReport(number, x, y)
            
        trial           = Struct()
        trial.position  = XY(x, y)
        trial.windSpeed = self.windSpeed
        return trial
            
    def summaryReport(self, history):
        Xmax = []
        Ymax = []
        wind = []
        for trial in history:
            Xmax.append(max(trial.position.x))
            Ymax.append(max(trial.position.y))
            wind.append(trial.windSpeed.get())
            pyplot.plot(trial.position.x, trial.position.y)
            
        print ('X Final  Range | Avg: {:.2f}m, Max: {:.2f}m, Min: {:.2f}m'
               .format(sum(Xmax)/len(Xmax), max(Xmax), min(Xmax)))
        print ('Y Height Range | Avg: {:.2f}m, Max: {:.2f}m, Min: {:.2f}m'
               .format(sum(Ymax)/len(Ymax), max(Ymax), max(0, min(Ymax))))
        pyplot.show()
            
    def run(self, verbose = False, silent = False):
        history = []
        for trial in range(1, self.numTrials + 1):
            history.append(self.runTrial(trial, verbose))
        
        if verbose or not silent: self.summaryReport(history)
        if silent: return history
